# Dog Disguising

Transform into a tamed wolf and fool your friends! This mod allows players to disguise themselves as a dog with synchronized health, movement, and animations.

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.4%20%26%201.21.5-brightgreen.svg)
![Java](https://img.shields.io/badge/Java-21-blue.svg)

## Commands
- `/disguise` - Turn into a wolf
- `/undisguise` - Return to Human

## How to Use
1. Install Fabric Loader for Minecraft 1.21.4 or 1.21.5.
2. Download and place the Dog Disguising mod in your `mods` folder.
3. Launch Minecraft with the Fabric profile.
4. Use the commands to disguise or return to your human form.
