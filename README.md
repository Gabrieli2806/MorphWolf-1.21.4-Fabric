# Dog Disguising

Transform into a tamed wolf and fool your friends! This mod lets you disguise yourself as a dog with synchronized health, movement, and animations. Commands:/disguise - Turn into a wolf /undisguise - Return to Human

![Platform](https://img.shields.io/badge/platform-Fabric-informational) ![Minecraft](https://img.shields.io/badge/minecraft-1.21.4-green)

## How to Use

1. Download the `.jar` from [Releases](https://github.com/Gabrieli2806/MorphWolf-1.21.4-Fabric/releases).
2. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft **1.21.4**.
3. Place the mod `.jar` in your `.minecraft/mods/` folder.
4. Launch Minecraft with the Fabric profile.
